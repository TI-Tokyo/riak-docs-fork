To be clear, for production deployments we recommend using no fewer
than 5 nodes:

http://docs.basho.com/riak/latest/references/appendices/Cluster-Capacity-Planning/#Number-of-Nodes

http://docs.basho.com/riak/latest/cookbooks/Backups/#LevelDB-Backups
