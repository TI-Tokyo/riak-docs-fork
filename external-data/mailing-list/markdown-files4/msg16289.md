&gt; FWIW, -noinput option of erl [1] makes beam not to read input and disables
&gt; interactive shell. Runner scripts that rebar generates passes extra args
&gt; of console command to erl (actually erlexec), one can add the option as:
&gt; 
&gt; riak console -noinput
&gt; 
&gt; Note: some features can not be used, e.g. pid file or "riak attach-direct".
&gt; 
&gt; [1] http://erlang.org/doc/man/erl.html
